# logic-web

> Integrated web platform repository for Team LOGIC's official homepage.

Team LOGIC의 공식 웹페이지 개발을 위한 통합 소스코드 저장소입니다.
본 프로젝트는 효율적인 협업과 안전한 버전 관리를 위해 Git / GitHub Workflow를 준수합니다.

git checkout main
git pull origin main

git checkout <본인이름>
git merge main

git add .
git commit -m "[Feat] 구현한 기능 요약 설명"
git push origin <본인이름>

!! 절대 main에 push하지 마세요.

Commit Rule
[Feat]: 새로운 기능 추가 시
[Fix]: 버그 및 오류 수정
[Design]: CSS 등 UI 디자인 변경 시
